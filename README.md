# HTML Games Collection

A curated collection of web-based games designed for toddlers (1+ years old). These games focus on simple interactions, vibrant visuals, and engaging feedback.

## Games in this repository

### [Keyboard Fireworks](1yo/keyboard-fireworks)
A fullscreen interactive experience where any key press creates a beautiful firework effect with letters, particles, and motion.
- **Goal**: Introduce the relationship between keyboard input and visual output.
- **Key Features**: Fullscreen mode, deterministic key mapping, multiple effect types, and accessible design.

### [Cat and Mouse](1yo/cat-and-mouse)
A simple game involving a cat and a mouse.
- **Status**: Basic implementation.

## Tech Stack
- **Languages**: HTML5, CSS3, JavaScript (ES6+).
- **Approach**: Static applications with zero runtime dependencies.
- **Visuals**: Modern aesthetics, smooth animations, and responsive design.

## How to use
You can run these games directly in your browser:
1. Clone the repository.
2. Open the index.html (or appropriate entry point) of the desired game in your browser.
3. For developers: Run 
pm install and 
pm run dev in specific game directories (like keyboard-fireworks) for a development environment.
