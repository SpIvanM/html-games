# Инструкции для агентов

Этот проект содержит систему автономной сборки «одного файла» для портативности.

## Критическое правило
**После любого изменения** логики в папке \src/\ или разметки в \index.html\, вы **ОБЯЗАНЫ** перезапустить скрипт сборки:

\\\powershell
node build.mjs
\\\

Это обновит файл \dist/index.html\, который является основным артефактом для запуска игры без сервера.

## Технические детали
- **Сборка**: Скрипт \uild.mjs\ инлайнит CSS и JS (удаляя ES-импорты/экспорты) прямо в HTML.
- **Тесты**: Всегда запускайте \
pm test\ перед сборкой.
